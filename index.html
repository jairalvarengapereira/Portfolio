Sobre Mim

Olá! Meu nome é Jair Alvarenga Pereira, sou analista desenvolvedor e apaixonado por tecnologia. Formado em Tecnologia em Informática, pelo Centro Universitário Newton Paiva, desde 2003, atualmente, estou cursando pós-graduação EAD em Ciência de Dados pela Facuminas. Com experiências que vão desde análise de suporte, análise de negócios bancários, entre outras, trabalho com desenvolvimento de sistemas desde 2007, utilizando linguagens como Delphi, Firemonkey, C++ Builder, PHP, NsBasic, entre outras. Atualmente, estou focado em desenvolvimento de sistemas web, utilizando as tecnologias: ReactJs, NodeJs, React Native, MongoDb entre outras.

Alguns de meus projetos

Agendei

Afetivograma

Aplicação mobile desenvolvida em Rad Studio Delphi XE 11.3 e SQLite, para avaliação de afetividade de pacientes em tratamentos de saúde mental.

CadUsuario

Aplicação CRUD para cadastro de usuários, desenvolvida em ReactJs com banco de dados MongoDB.
